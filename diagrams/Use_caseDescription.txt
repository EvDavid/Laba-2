Прецедент: Нажата кнопка «Выбор вакансии»
* Краткое описание: Вариант использования «Выбор вакансии» позволяет пользователю найти работу.
* Предусловие: Запуск приложения пользователем.
* Основной поток событий: Пользователь нажал кнопку «Выбор вакансии», ему предлагается подать резюме, либо откликнуться на объявление.
* Постусловие: Пользователь выбрал нужное объявление из списка и ему выводится полная информация, о данной вакансии.
Прецедент: Нажата кнопка «Поиск»
* Краткое описание: Вариант использования «Поиск» производит поиск профессии.
* Предусловие: Запуск приложения пользователем.
* Основной поток событий: Пользователь нажал на кнопку «Поиск», далее ему необходимо выбрать понравившуюся работу для поиска.
* Постусловие: Если пользователь ввел корректную профессию, то ему выводится список последних вакансий по этой профессии.
Прецедент: Нажата кнопка «Фильтра»
* Краткое описание: Вариант использования «Настройка фильтров» позволяет пользователю установить уровень оплаты, выбор города, график, занятость.
* Предусловие: Запуск приложения пользователем.
* Основной поток событий: Пользователь нажал кнопку «Настройки фильтров» и установил значения вышеназванных настроек по своему усмотрению.
* Постусловие: После изменения настроек пользователю предоставляется возможность продолжить работу с приложением.
